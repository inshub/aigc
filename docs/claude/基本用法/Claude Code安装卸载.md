# Claude Code安装卸载
[TOC]




### 前提条件
确保MacOS 的Node.js 版本 ≥ 18.0


### 安装Claude Code
```
#全局安装
npm install -g @anthropic-ai/claude-code

claude --version

#检查安装路径
npm ls -g  @anthropic-ai/claude-code
```

### 更新 Claude Code
```
claude update
```


### 卸载Claude Code
```
npm uninstall -g @anthropic-ai/claude-code
```
